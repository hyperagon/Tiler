# Tiler
A Tiled Mesh SVG Builder, vibe-coded in 2025-09-24.

Number of prompts: Around 18+.

- No Dependencies
- Pyhon

## Arguments
- **width** and **height** of the **document**
- how many **corners** each Tile has (3,4 or 6)
- **fill** and **stroke color**
- **stroke width**
- **Tile size (width=height)**
